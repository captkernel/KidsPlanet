# Kids Planet Website — Improvement Research (March 2026)

## Gap Analysis: What Top School Websites Have That We're Missing

### HIGH PRIORITY (Trust & Conversion)
1. **Staff/Faculty Directory** — teacher profiles with photos, qualifications, bio
2. **Fee Structure Page** — even a simplified table with "Contact for details"
3. **Downloadable Curriculum PDFs** — one-page overview per program level
4. **Video Content** — school tour or "Day at Kids Planet" video
5. **Event Calendar** — structured calendar vs flat announcement list

### MEDIUM PRIORITY (Engagement)
6. **Blog / News Section** — SEO-boosting content like "Why Outdoor Play Matters"
7. **Virtual Tour / Photo Walkthrough** — guided tour of classrooms, playground
8. **FAQ Section** — common questions about admissions, transport, timings
9. **Awards / Achievements** — student competition wins, results
10. **Google Review Integration** — embed real reviews for third-party credibility

### LOWER PRIORITY (Polish)
11. Accessibility features (text size, contrast)
12. Audience-based navigation ("New Parent? Start Here")
13. Full online admission application (not just inquiry)
14. Parent portal (homework, attendance — needs backend)
15. Transport/bus route info
16. Safety & infrastructure details (CCTV, fire safety)
17. Daily routine / schedule display

## UX Improvements Needed
- Click-to-call phone number in header (mobile)
- WhatsApp pre-filled message: "Hi, I'd like to know about admissions"
- Social proof near CTAs (stats/testimonials near inquiry buttons)
- Sticky "Apply Now" CTA on all pages

## Available MCP Servers for Implementation
- **Unsplash/Stock Images MCP** — real stock photos for placeholders
- **MCP Image Optimizer** — optimize images for mobile performance
- **Next.js DevTools MCP** — built into Next.js 16 at /_next/mcp
- **Firecrawl** — scrape competitor school sites for inspiration
- **Bright Data Facebook/Instagram** — pull social media (requires API key)
